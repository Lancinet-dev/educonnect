# EduConnect

SaaS scolaire multi-écoles pour les écoles guinéennes (Afrique de l'Ouest).

Connecte sur une seule plateforme : Fondateur, Directeurs, Censeurs,
Surveillants, Comptables, Enseignants, Élèves et Parents.

## Stack

- **Frontend** : React 18 + Vite 5 + Tailwind CSS v3 + React Router v6 + Zustand + TanStack React Query v5
- **Backend** : Node.js + Express 4 + PostgreSQL (pg) + JWT + Socket.IO
- **Base de données** : PostgreSQL, migrations SQL numérotées (`database/migrations/`)

## Démarrage

```bash
# Backend
cd backend && npm install && npm run dev

# Frontend
cd frontend && npm install && npm run dev
```

- API : http://localhost:5000
- App : http://localhost:5173
